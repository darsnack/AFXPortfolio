# AFX Portfolio
A repository to contain our audio effects portfolio for a class at Rose-Hulman.

# Portfolio
The portfolio PDF can be found under Portfolio/main.pdf. All the links in the portfolio work relative to this location.

# Directory Structure
- At the root directory contains all MATLAB code
- The InputAudio folder contains all input audio samples
- The OutputAudio folder contains all output audio samples
- The Portfolio folder contains all the files connected to the portfolio
